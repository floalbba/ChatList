ChatList

Один промт — несколько нейросетей. Сравнивайте ответы в одном окне.

Скачать для Windows

Версия 1.0.0

Возможности

  • Отправка промта в несколько моделей одновременно (OpenRouter, OpenAI, DeepSeek, Groq)
  • AI-ассистент для улучшения и переформулировки промтов
  • Сохранение промтов и результатов в локальную базу
  • Экспорт выбранных ответов в Markdown или JSON
  • Настройки: светлая/тёмная тема, размер шрифта

Требования

  • Windows 10 или 11
  • API-ключ OpenRouter (или другой поддерживаемый провайдер)

Установка

  1. Скачайте ChatList-X.X.X-Setup.exe из раздела Releases
  2. Запустите установщик
  3. Создайте файл .env в папке приложения с переменной OPENROUTER_API_KEY=ваш_ключ
