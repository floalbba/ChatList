# ChatList 1.0.0

## Описание

ChatList — программа для отправки одного промта в несколько нейросетей и сравнения их ответов.

## Что нового

- Отправка промта в несколько моделей (OpenRouter и др.)
- AI-ассистент для улучшения промтов
- Сохранение промтов и результатов
- Экспорт в Markdown/JSON
- Настройки: тема, размер шрифта

## Установка

1. Скачайте **ChatList-1.0.0-Setup.exe**
2. Запустите установщик
3. Добавьте API-ключ OpenRouter в `.env` (см. `.env.example`)

## Требования

- Windows 10/11
- API-ключ OpenRouter (https://openrouter.ai)
